Introduction

Duct tape is a simple responsive grid for websites. It is meant to be simple and intuitive to use to create nice-looking responsive web layouts for your projects.

Duct tape doesn't claim to be the last responsive grid system you will ever use; instead we want it to be the first, your starting for developing your own responsive grid systems.

You can use the duct tape grid as a base for your own grids and layout engine, it is customizable and can adapt to have as many columns as you like.

Build with SASS

To build the the styles for this demo page run this in your command line: 

$ # build continuously, nested, no-cache
$ sass --watch -t nested -C _main.scss:build.css
$
$ # build once, nested, no-cache
$ sass --update -t nested -C _main.scss:build.css

Classes

Duct tape's grid is meant to be used out-of-the-box, as they say. Every class can be used as a default if no other class is used in combination. This meand that if a <div> only needs one class to define its layout, you don't have to add all three classes for different screen sizes to it.

Forms

Forms in Duct Tape are pretty and simple.

A form should be wrapped in a cage so you can use columns to create margins around the input fields. On the other hand field sets should be inside a column so they display properly.
